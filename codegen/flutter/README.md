# flutter codegen

이 폴더는 Flutter 타깃 codegen의 사람용 엔트리 문서다.

공통 codegen baseline은 [`../CODEGEN_BASELINE.md`](../CODEGEN_BASELINE.md) 를 먼저 따른다.

## Structure

- `guide/README.md`: Flutter guide 폴더 개요
- `guide/IR_CODEGEN_BASELINE.md`: `flutter_ir` 해석 baseline
- `guide/FLUTTER_APP_BASELINE.md`: Flutter 앱 구조 baseline
- `guide/GUIDE.md`: 세부 생성 규칙
- `guide/CHECKLIST.md`: 생성 결과 검증 기준
- `guide/PROMPT_TEMPLATE.md`: 코드 생성 프롬프트 템플릿
- `guide/USER_REQUEST_TEMPLATE.md`: 사용자 입력 템플릿
